Tutorials
=========

End-to-end guides that show how wraquant's modules work together for
real quantitative finance workflows. Each tutorial walks through a
complete pipeline with runnable code and interpretation of results.

.. grid:: 2

   .. grid-item-card:: Risk Analysis
      :link: risk_analysis
      :link-type: doc

      Compute risk metrics, VaR, stress test against historical crises,
      decompose portfolio risk, and generate a risk report.

   .. grid-item-card:: Regime-Based Investing
      :link: regime_investing
      :link-type: doc

      Detect market regimes with HMM, analyze per-regime statistics,
      build a regime-conditional portfolio, and backtest vs buy-and-hold.

   .. grid-item-card:: Volatility Modeling
      :link: volatility_modeling
      :link-type: doc

      Fit GARCH models, compute news impact curves, compare model variants,
      forecast volatility, and run rolling GARCH.

   .. grid-item-card:: Portfolio Construction
      :link: portfolio_construction
      :link-type: doc

      Optimize with MVO, risk parity, and Black-Litterman, decompose risk
      contributions, and adjust allocations for market regimes.

   .. grid-item-card:: Backtesting Strategies
      :link: backtesting_strategies
      :link-type: doc

      Define a strategy, backtest with the event-driven engine, analyze
      metrics, generate tearsheets, and run walk-forward validation.

   .. grid-item-card:: ML Alpha Research
      :link: ml_alpha_research
      :link-type: doc

      Engineer features from TA indicators, train with purged CV,
      walk-forward validate, evaluate with financial metrics, and
      track experiments.
